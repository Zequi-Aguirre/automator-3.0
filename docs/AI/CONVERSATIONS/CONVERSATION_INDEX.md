### **How to Add a New Entry**
1. Follow the naming convention from `/docs/AI/CONVERSATION_ARCHIVING_PROTOCOL.md`.
2. Create the conversation Markdown file under `/docs/AI/CONVERSATIONS/{ProjectName}/`.
3. Add an entry in this index with:
   - Conversation ID
   - Project name
   - Topic title
   - Date
   - Short summary
   - Related files or modules
